# 🎥 Remote Interview Platform

A modern, real-time collaborative platform for conducting remote technical interviews. Built with cutting-edge technologies for seamless video collaboration, live code editing, and instant communication.

## ✨ Features

- **Live Video Conferencing** - Crystal-clear video and audio streaming powered by Stream.io
- **Real-Time Code Editor** - Monaco Editor integration for collaborative coding sessions
- **Modern UI/UX** - Beautiful, responsive interface with Radix UI components and Tailwind CSS
- **User Authentication** - Secure authentication with Clerk
- **Dark Mode Support** - Theme switching with next-themes
- **Database Integration** - Convex for real-time data synchronization
- **Webhook Support** - Svix for reliable event handling
- **Resizable Panels** - Flexible layout with draggable panels for custom workspace arrangements

## 🛠️ Tech Stack

- **Frontend Framework**: [Next.js 14](https://nextjs.org/) - React with server-side rendering
- **Language**: [TypeScript](https://www.typescriptlang.org/) - Type-safe development
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS
- **UI Components**: [Radix UI](https://www.radix-ui.com/) - Unstyled, accessible components
- **Code Editor**: [Monaco Editor](https://microsoft.github.io/monaco-editor/) - VS Code's editor
- **Video/Streaming**: [Stream.io Video React SDK](https://getstream.io/) - Real-time video communication
- **Authentication**: [Clerk](https://clerk.com/) - Modern auth solution
- **Backend/Database**: [Convex](https://www.convex.dev/) - Real-time database
- **Icons**: [Lucide React](https://lucide.dev/) - Beautiful icon library
- **Notifications**: [React Hot Toast](https://react-hot-toast.com/) - Non-intrusive notifications
- **Event Handling**: [Svix](https://www.svix.com/) - Webhooks as a service
- **Date Utilities**: [date-fns](https://date-fns.org/) - Functional date manipulation
